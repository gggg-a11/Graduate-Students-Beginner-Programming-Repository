1) 盒模型与布局

属性作用示例
box-sizing 控制宽高是否包含内边距与边框 box-sizing: border-box;
display 元素显示类型（块、内联、弹性、网格…） display: block | inline | inline-block | flex | grid;
width / height 设置宽高，可用 px/%/vw/vh 等单位 width: 300px; height: 200px;
max-width / min-width 限制尺寸范围，响应式常用 max-width: 800px;
margin 外边距，控制元素之间间隔/居中 margin: 20px auto;（水平居中）
padding 内边距，内容与边框的距离 padding: 12px 16px;
border 边框宽度、样式、颜色 border: 1px solid #ddd;
border-radius 圆角 border-radius: 8px;
box-shadow 盒子阴影 box-shadow: 0 4px 12px rgba(0,0,0,.12);

示例：卡片容器

标题

这是一个使用 box-model 常规属性构建的卡片。

<div style="
  box-sizing: border-box;
  max-width: 360px;
  margin: 0 auto;
  padding: 16px;
  border: 1px solid #e5e7eb;
  border-radius: 12px;
  box-shadow: 0 6px 20px rgba(0,0,0,.08);
">卡片内容</div>

2) 定位与溢出

属性作用示例
position 定位模型：static/relative/absolute/fixed/sticky position: sticky; top: 0;（吸顶）
top/right/bottom/left 配合定位偏移 top: 10px; left: 10px;
z-index 层叠顺序（定位元素常用） z-index: 1000;
overflow 内容溢出处理 overflow: hidden | auto | scroll;

示例：固定右下角返回顶部按钮

滚动页面时按钮固定在右下角（position: fixed）。

<button style="
  position: fixed;
  right: 18px;
  bottom: 18px;
  padding: 8px 12px;
  background: #0b6bcb; color: #fff; border: 0; border-radius: 8px;
">↑ 顶部</button>

3) 弹性布局（Flexbox）

属性作用示例
display: flex 启用弹性布局 display: flex;
justify-content 主轴对齐 justify-content: space-between;
align-items 交叉轴对齐 align-items: center;
gap 子项间距（优雅替代 margin 间隔） gap: 12px;
flex 子项伸缩（flex: 1 平分空间） flex: 1;
flex-wrap 换行 flex-wrap: wrap;

示例：等分三列自适应

A
B
C
<div style="display:flex; gap:10px">
  <div style="flex:1">A</div>
  <div style="flex:1">B</div>
  <div style="flex:1">C</div>
</div>

4) 网格布局（Grid）

属性作用示例
display: grid 启用网格布局 display: grid;
grid-template-columns 定义列轨道 grid-template-columns: repeat(3, 1fr);
gap 网格间距 gap: 12px;
place-items 网格项对齐 place-items: center;

示例：自适应卡片网格

1
2
3
4
<div style="display:grid;gap:10px;grid-template-columns:repeat(auto-fit,minmax(120px,1fr))">
  <div>1</div> ... 
</div>

5) 文本与字体

属性作用示例
color文字颜色color: #333;
font-size字体大小font-size: 16px;
font-weight粗细font-weight: 700;
line-height行高line-height: 1.6;
text-align对齐text-align: center;
text-decoration下划线等text-decoration: none;
letter-spacing字距letter-spacing: 0.5px;
word-break单词/中文换行word-break: break-all;
white-space空白处理white-space: nowrap;
text-overflow省略号（需配合）overflow:hidden;white-space:nowrap;text-overflow:ellipsis;

示例：单行溢出省略号

这是一段很长很长很长的标题，用于演示单行文本溢出时的省略号效果…… 
<div style="
  width:220px; overflow:hidden; white-space:nowrap; text-overflow:ellipsis;
">很长的标题……</div>

6) 背景

属性作用示例
background-color背景色background-color:#f8fafc;
background-image背景图background-image:url(bg.jpg);
background-size尺寸background-size: cover | contain;
background-repeat平铺background-repeat: no-repeat;
background-position定位background-position: center;
linear-gradient渐变background: linear-gradient(45deg,#60a5fa,#34d399);

示例：渐变背景横幅

渐变 Banner 
<div style="
  background: linear-gradient(90deg,#60a5fa,#34d399);
  color:#fff; padding:16px; text-align:center; border-radius:10px;
">渐变 Banner</div>

7) 图片与媒体

属性作用示例
max-width响应式图片宽度max-width: 100%;
height常与 auto 保持比例height: auto;
object-fit填充裁切策略object-fit: cover | contain;
aspect-ratio固定宽高比aspect-ratio: 16/9;
filter滤镜filter: grayscale(100%);

示例：固定 16:9 比例封面

<div style="aspect-ratio:16/9; overflow:hidden">
  <img src="image.jpg" style="width:100%; height:100%; object-fit:cover">
</div>

8) 交互与动画

属性作用示例
cursor鼠标指针cursor: pointer;
transition过渡动画transition: all .2s ease;
transform位移/缩放/旋转transform: translateY(-2px);
opacity透明度opacity: .8;
user-select文本可选user-select: none;
outline焦点可视outline: 2px solid #60a5fa;

示例：有悬停动效的按钮

<button style="
  cursor:pointer; transition: transform .18s ease, box-shadow .18s ease;
">按钮</button>

9) 响应式技巧

移动端页面请确保有视口：<meta name="viewport" content="width=device-width, initial-scale=1">

媒体查询

@media (max-width: 600px) {
  .card { padding: 12px; }
}

流式尺寸与函数

/* clamp(最小值, 首选值, 最大值) */
h1 { font-size: clamp(20px, 4vw, 32px); }

10) 常用布局模板（可直接用）

（1）居中容器

<div style="max-width:860px;margin:0 auto;padding:16px">内容</div>

（2）顶部导航

<header style="position:sticky;top:0;background:#fff;border-bottom:1px solid #eee;z-index:10">
  <nav style="display:flex;align-items:center;justify-content:space-between;max-width:960px;margin:0 auto;padding:12px 16px">
    <strong>Logo</strong>
    <div style="display:flex;gap:12px">
      <a href="#">首页</a><a href="#">文档</a><a href="#">关于</a>
    </div>
  </nav>
</header>

（3）两栏布局（主内容 + 侧栏）

<div style="display:grid;grid-template-columns:1fr 280px;gap:16px">
  <main>主内容</main>
  <aside>侧栏</aside>
</div>

（4）卡片

<article style="border:1px solid #e5e7eb;border-radius:12px;padding:16px;box-shadow:0 8px 24px rgba(0,0,0,.06)">
  <h3 style="margin:0 0 8px">卡片标题</h3>
  <p style="margin:0">卡片内容</p>
</article>

（5）页脚

<footer style="margin-top:30px;padding:20px;text-align:center;color:#666;border-top:1px solid #eee">
  © 2025 你的站点
</footer>
